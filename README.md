# RF ONLINE NEXT Strategy Guide

A comprehensive bilingual (EN/ID) strategy guide website for Netmarble's sci-fi MMORPG **RF ONLINE NEXT**.

## Features

- **6 Pages**: Home, Races & Classes, Beginner Guide, Equipment, PVP Warfare, Farming Guide
- **Bilingual**: English & Indonesian with one-click language toggle
- **SEO Optimized**: Per-page meta tags, Open Graph, Twitter Cards, Schema.org structured data
- **Responsive**: Dark sci-fi theme, mobile-friendly layout
- **AI-generated game art**: Custom banners and icons for each section

## Pages

| Page | Description |

| Home | Game overview, core features, news, strategy navigation |

- Pure HTML/CSS/JS — no frameworks, zero dependencies
